Sales Performance Dashboard

A business intelligence dashboard project using SQL, Excel, and Power BI to clean sales data, calculate KPIs, and turn raw transactions into business insights.

GitHub Repository Live Demo

Case Study

Problem and solution

Challenge

Raw sales data is often messy and difficult for business users to interpret. The goal is to transform it into a dashboard that clearly explains performance and growth.


Approach

The project uses SQL for cleaning and transformation, Excel for validation where needed, and Power BI for KPI reporting and interactive business dashboards.


Key Features

  • SQL data cleaning and validation
  • Revenue, order, and customer KPI calculations
  • Average order value and monthly growth analysis
  • Power BI dashboard layout
  • Business reporting structure
  • Portfolio-ready analytics case study

Architecture

  • Raw CSV or database tables are cleaned and validated
  • SQL queries calculate KPIs and reporting views
  • Power BI connects to transformed data for dashboard visuals
  • Final report explains insights, trends, and recommendations
