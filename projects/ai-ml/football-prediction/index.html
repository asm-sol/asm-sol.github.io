Football Match Prediction System

A football prediction engine using Elo ratings, statistical calibration, ML modelling, and value betting analysis for match outcome evaluation.

GitHub Repository Live Demo

Case Study

Problem and solution

Challenge

Sports prediction requires calibrated probabilities, not just predicted winners. A model must understand team strength, market prices, historical form, and value thresholds.


Approach

The system combines Elo-style team ratings, Dixon-Coles inspired modelling, machine learning features, probability calibration, and backtesting to evaluate whether predictions create real market edge.


Key Features

  • Elo-based team strength modelling
  • Dixon-Coles style calibration planning
  • Machine learning prediction pipeline
  • Market odds and value betting analysis
  • Backtesting and model evaluation
  • Discord alert distribution planning

Architecture

  • Data collector loads fixtures, results, odds, and league history
  • Feature pipeline creates team strength, form, and market variables
  • Model layer generates calibrated match probabilities
  • Alert layer distributes high-edge selections to Discord
