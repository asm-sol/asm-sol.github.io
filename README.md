# Alexander Stuard Moolman Portfolio

GitHub Pages portfolio website.

## How to use

1. Create a GitHub repo named `YOURUSERNAME.github.io`
2. Upload these files to the repo
3. Go to Settings → Pages
4. Set Source to `Deploy from a branch`
5. Select `main` and `/root`
6. Visit `https://YOURUSERNAME.github.io`

## CV

Place your CV PDF inside:

`assets/cv.pdf`

Then the CV download button will work.
